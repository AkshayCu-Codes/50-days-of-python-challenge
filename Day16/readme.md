# Day 16: Sum the List



## ✅ Challenge

### 🔹 Problem Statement  
Write a function called `sum_list` that takes **one parameter**, a **nested list of integers**, and returns the **sum of all integers** in the list.

---

### ✏️ Example

```python
sum_list([[2, 4, 5, 6], [2, 3, 5, 6]])    # Output: 33
sum_list([[1], [2, 3], [4]])             # Output: 10
